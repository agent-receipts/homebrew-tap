# agent-receipts/homebrew-tap

Homebrew tap for [agent-receipts](https://github.com/agent-receipts) tools.

## Usage

```sh
brew install agent-receipts/tap/mcp-proxy
```

The formula here is automatically updated by GoReleaser on every `mcp-proxy/vX.Y.Z` release in [agent-receipts/ar](https://github.com/agent-receipts/ar).
